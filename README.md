# Olea Computer Website

Welcome to the Olea Computer website project! This project is built using Next.js and aims to showcase our innovative products in the consumer electronics space, focusing on AI electronics, custom silicon, and beautifully designed operating systems.

## Project Structure

The project is organized as follows:

```
olea-computer-website
├── pages
│   ├── _app.js          # Initializes pages and adds global styles
│   ├── _document.js     # Customizes the HTML structure and includes fonts
│   └── index.js         # Main entry point for the homepage
├── public
│   ├── images           # Contains product images and branding visuals
│   └── favicon.ico      # Favicon for the website
├── styles
│   └── globals.css      # Global CSS styles for a minimal aesthetic
├── package.json         # npm configuration file
├── next.config.js       # Custom Next.js configuration
└── README.md            # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd olea-computer-website
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the development server:**
   ```
   npm run dev
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```

## Technologies Used

- **Next.js**: A React framework for building server-side rendered applications.
- **React**: A JavaScript library for building user interfaces.
- **CSS**: For styling the components and ensuring a clean, minimal aesthetic.
